# 🔧 Nicmah System Troubleshooting Guide

## 🚨 Quick Fixes for Common Problems

---

## 📱 **Browser Issues**

### **Page Won't Load**
**Problem**: Page shows "Cannot connect" or blank screen
**Quick Fix**:
1. Refresh the page (F5 or Ctrl+R)
2. Check internet/network connection
3. Try a different browser
4. Clear browser cache and cookies

**If Still Not Working**:
- Contact IT support
- Check if server is running
- Verify correct URL

### **Slow Loading**
**Problem**: Pages take too long to load
**Quick Fix**:
1. Close other browser tabs
2. Clear browser cache
3. Check internet speed
4. Try different browser

---

## 🔐 **Login Problems**

### **Can't Log In**
**Problem**: Username/password not working
**Quick Fix**:
1. Check Caps Lock is OFF
2. Verify username spelling
3. Try typing password slowly
4. Check for extra spaces

**If Still Not Working**:
- Contact administrator for password reset
- Verify account is active
- Check if account is locked

### **Forgot Password**
**Problem**: Don't remember password
**Solution**:
- Contact system administrator immediately
- Do NOT try to guess multiple times
- Administrator will reset password

---

## 🛒 **Sales Recording Issues**

### **Can't Add Products to Sale**
**Problem**: Products not appearing in dropdown
**Quick Fix**:
1. Check if product exists in system
2. Verify product is active
3. Check if product has stock
4. Try refreshing the page

**If Still Not Working**:
- Contact administrator to check product setup
- Verify product categories are configured

### **VAT Calculations Wrong**
**Problem**: Tax amounts seem incorrect
**Quick Fix**:
1. Check product VAT rate in product list
2. Verify quantities entered correctly
3. Check if discount is applied properly
4. Refresh page and try again

**If Still Not Working**:
- Contact administrator to verify VAT settings
- Check if product prices are updated

### **Sale Won't Save**
**Problem**: "Record Sale" button not working
**Quick Fix**:
1. Check all required fields are filled
2. Verify seller name is entered
3. Ensure at least one product is added
4. Check for error messages

**If Still Not Working**:
- Try refreshing the page
- Check if products have sufficient stock
- Contact administrator if problem persists

---

## 📦 **Product Issues**

### **Can't Find Products**
**Problem**: Products not showing in search
**Quick Fix**:
1. Check spelling in search box
2. Try different search terms
3. Clear search and browse all products
4. Check product category filters

**If Still Not Working**:
- Contact administrator to verify product setup
- Check if products are active

### **Stock Numbers Wrong**
**Problem**: Stock levels seem incorrect
**Quick Fix**:
1. Check recent sales recordings
2. Verify if products were restocked
3. Check for stock adjustments
4. Refresh the page

**If Still Not Working**:
- Contact administrator for stock audit
- Check if there are pending transactions

4. Check if sales were recorded

**If Still Not Working**:
- Contact administrator to verify data
- Check if reports are configured properly

### **Can't Export Reports**
**Problem**: Export button not working
**Quick Fix**:
1. Check if report has data
2. Verify file format selection
3. Check browser download settings
4. Try different file format

**If Still Not Working**:
- Check browser popup blockers
- Verify sufficient disk space
- Contact administrator

## 💾 **Data Export Issues**

### **Export File Corrupted**
**Problem**: Downloaded file won't open
**Quick Fix**:
1. Check file size (should not be 0 bytes)
2. Try downloading again
3. Check file extension (.xlsx or .csv)
4. Try different file format

**If Still Not Working**:
- Contact administrator
- Check if export function is working
- Verify sufficient data exists

### **Export Takes Too Long**
**Problem**: Export process hangs
**Quick Fix**:
1. Wait for process to complete
2. Check if large amount of data
3. Try smaller date ranges
4. Cancel and try again

**If Still Not Working**:
- Contact administrator
- Check system performance
- Try during off-peak hours

---

## 🖥️ **System Performance Issues**

### **System Very Slow**
**Problem**: Everything takes too long to respond
**Quick Fix**:
1. Close other applications
2. Check internet connection
3. Try refreshing page
4. Log out and log back in

**If Still Not Working**:
- Contact administrator
- Check if system is overloaded
- Try during off-peak hours

### **System Crashes**
**Problem**: System stops responding
**Quick Fix**:
1. Wait 1-2 minutes for recovery
2. Refresh the page
3. Log out and log back in
4. Clear browser cache

**If Still Not Working**:
- Contact administrator immediately
- Check if others are experiencing same issue
- Document what you were doing when it crashed

---

## 📞 **Getting Help**

### **When to Contact Administrator**
- **Immediate**: System crashes, data loss, security issues
- **Same Day**: Can't log in, critical functions not working
- **Next Day**: Performance issues, minor bugs

### **Information to Provide**
1. **What you were trying to do**
2. **What happened instead**
3. **Error messages (if any)**
4. **Your username**
5. **Time the problem occurred**
6. **Steps to reproduce the problem**

### **Contact Methods**
- **Urgent**: Phone call to administrator
- **Normal**: Email or system message
- **General**: Team meeting or training session
