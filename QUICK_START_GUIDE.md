# 🚀 Nicmah System Quick Start Guide

## ⚡ Get Up and Running in 5 Minutes

### 1. **Access the System**
- Open your web browser
- Go to: `http://localhost:8000` (or your server address)
- You'll be redirected to login

### 2. **First Time Setup**
- **Access Admin Panel**: Go to `http://localhost:8000/admin/`
- **Login** with your admin credentials
- **Create Users** for your staff members
- **Add Products** to your inventory

### 3. **Essential Daily Tasks**

#### Morning (5 minutes)
- ✅ Check dashboard for alerts
- ✅ Review low stock items
- ✅ Check expiry warnings

#### During Sales (as needed)
- ✅ Record sales immediately
- ✅ Check stock levels
- ✅ Monitor real-time metrics

#### Evening (10 minutes)
- ✅ Record any remaining sales
- ✅ Review daily performance
- ✅ Plan restocking

### 4. **Key Features to Use**

| Feature | When to Use | How to Access |
|---------|-------------|---------------|
| **Dashboard** | Every morning | Main page after login |
| **Sales Entry** | During/after sales | Sales → New Sale |
| **Products** | Check stock | Products menu |
| **Reports** | End of day/week | Reports menu |
| **Analytics** | Weekly review | Analytics menu |

### 5. **Quick Commands**

#### Record a Sale
1. Click "Record New Sale"
2. Enter seller name
3. Add products + quantities
4. Click "Record Sale"

#### Check Stock
1. Go to Products
2. Use search box
3. Look at stock levels (color-coded)

#### Generate Report
1. Go to Reports
2. Select date range
3. Choose report type
4. Click "Generate"

### 6. **Color Codes to Remember**
- 🟢 **Green**: Good/Healthy
- 🟠 **Orange**: Warning/Attention needed
- 🔴 **Red**: Critical/Action required

- **User Manual**: Read `USER_MANUAL.md`
