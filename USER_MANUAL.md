## 🚀 Getting Started

### First Time Login
1. **Open your web browser** (Chrome, Firefox, Edge, or Safari)
2. **Navigate to**: `http://localhost:8000` (or your server address)
3. **Login** with your username and password
4. **You'll see the main dashboard** with all your business metrics

### System Requirements
- **Internet Browser**: Any modern browser (Chrome recommended)
- **Screen Resolution**: Minimum 1024x768 (works on tablets and mobile)
- **Network**: Local network access (works offline)

## 📦 Product Management

### Viewing Products
1. **Go to Products** from the main menu
2. **Search**: Use the search box to find specific products
3. **Filter**: Filter by category, stock status, or activity
4. **Sort**: Click column headers to sort by name, price, stock, etc.

### Product Information
Each product shows:
- **Name & SKU**: Product identification
- **Category**: Product classification
- **Current Stock**: Available quantity
- **Selling Price**: Current selling price (including VAT)
- **VAT Rate**: Tax percentage for the product
- **Stock Status**: In Stock / Low Stock / Out of Stock (color-coded)

### Product Details
Click any product to see:
- **Complete product information**
- **Batch details** with expiry dates
- **Sales history** for this product
- **Stock movements** and adjustments

### Managing Stock Levels
- **Green**: Good stock levels
- **Orange**: Low stock (needs reordering)
- **Red**: Out of stock (urgent reorder needed)

## 🛒 Sales Management

### Recording a New Sale
1. **Click "Record New Sale"** or go to Sales → New Sale
2. **Enter Seller Information**:
   - Seller Name (required)
   - Sale Type: Cash, Credit, M-Pesa, or Bank Transfer
3. **Add Products**:
   - Click "Add Item" to add products to the sale
   - Select product from the list
   - Enter quantity sold
   - Unit price is automatically filled (can be adjusted)
   - VAT is calculated automatically
4. **Review Totals**:
   - Subtotal (excluding VAT)
   - Total VAT amount
   - Total amount (including VAT)
5. **Add Discount** if applicable
6. **Add Notes** for any special information
7. **Click "Record Sale"** to complete

### Understanding VAT Calculations
- **Subtotal**: Price before tax (Quantity × Unit Price)
- **VAT Amount**: Tax calculated automatically per product
- **Total**: Final amount including all taxes
- **VAT rates** can be different per product (default 16%)

### Viewing Sales History
1. **Go to Sales** from the main menu
2. **View all sales** in chronological order
3. **Search** by invoice number or seller name
4. **Filter** by date range or sale type
5. **Click any sale** to see detailed breakdown

### Sales Information
Each sale shows:
- **Invoice Number**: Unique sale identifier
- **Date & Time**: When the sale was recorded
- **Seller Name**: Who made the sale
- **Total Amount**: Final sale value
- **Payment Method**: How customer paid
- **Items Count**: Number of different products sold

## 📥 Data Export

### Exporting Data
**Access**: Click "Export Data" in the main menu

#### Export Options
1. **Data Types**:
   - Products: Complete product catalog
   - Sales: Transaction history
   - Stock Levels: Current inventory status
   - Batches: Expiry and supplier information
   - All Data: Complete system backup

2. **File Formats**:
   - **Excel (.xlsx)**: Best for analysis and reporting
   - **CSV**: Compatible with other systems

#### Export Process
1. **Select data type** you want to export
2. **Choose file format** (Excel recommended)
3. **Set date range** if applicable
4. **Click "Export"** to download file
5. **File will download** to your computer

#### Uses for Exported Data
- **Backup**: Keep copies of your data
- **Analysis**: Use in Excel for advanced analysis
- **Reporting**: Create custom reports
- **Integration**: Import into other systems

## 🔧 Troubleshooting

### Common Issues & Solutions

#### Login Problems
**Issue**: Cannot log in
**Solutions**:
- Check username and password spelling
- Ensure Caps Lock is off
- Contact administrator for password reset
- Try refreshing the browser page

#### Page Not Loading
**Issue**: Pages load slowly or not at all
**Solutions**:
- Check internet/network connection
- Refresh the browser page (F5 or Ctrl+R)
- Clear browser cache and cookies
- Try a different browser
- Contact IT support if problem persists

#### Sales Not Recording
**Issue**: Cannot record new sales
**Solutions**:
- Ensure all required fields are filled
- Check that products are in stock
- Verify seller name is entered
- Check for any error messages
- Try refreshing and starting over

#### Reports Not Generating
**Issue**: Reports show no data or errors
**Solutions**:
- Check date range selection
- Ensure there's data for selected period
- Try different report type
- Refresh browser and try again
- Contact support if problem continues

#### Stock Numbers Wrong
**Issue**: Stock levels seem incorrect
**Solutions**:
- Check recent sales recordings
- Review stock adjustments in admin
- Verify batch quantities
- Check for duplicate entries
- Contact administrator for stock audit

### Getting Help
1. **Check this manual** first for answers
2. **Ask a colleague** who knows the system
3. **Contact your administrator** for account issues
4. **Contact IT support** for technical problems
5. **Keep error messages** to help with troubleshooting

## 💡 Tips & Best Practices

### Recording Sales
✅ **Do**:
- Record sales immediately or at least daily
- Double-check quantities and prices
- Use clear, consistent seller names
- Add notes for special circumstances
- Verify VAT calculations look correct

❌ **Don't**:
- Leave sales unrecorded for days
- Use inconsistent seller name spellings
- Forget to check stock levels
- Skip adding important sale details

### Managing Products
✅ **Do**:
- Keep product information up to date
- Monitor expiry dates regularly
- Set appropriate reorder levels
- Use consistent category names
- Check VAT rates are correct

❌ **Don't**:
- Let products expire without notice
- Ignore low stock warnings
- Use confusing product names
- Mix up different product variants

### Using Reports
✅ **Do**:
- Generate reports regularly
- Use appropriate date ranges
- Export important reports for backup
- Share reports with management
- Act on report insights

❌ **Don't**:
- Ignore report recommendations
- Use outdated report data
- Forget to backup important reports
- Overlook trends and patterns

### System Security
✅ **Do**:
- Log out when finished
- Keep login credentials secure
- Report suspicious activity
- Use strong passwords
- Backup data regularly

❌ **Don't**:
- Share login credentials
- Leave system unattended while logged in
- Ignore security warnings
- Delete important data without backup

### Data Management
✅ **Do**:
- Export data regularly for backup
- Keep consistent naming conventions
- Verify data accuracy regularly
- Clean up old unnecessary data
- Document important changes

❌ **Don't**:
- Rely only on system for data storage
- Make bulk changes without backup
- Ignore data inconsistencies
- Delete data without confirmation
